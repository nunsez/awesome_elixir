# AwesomeElixir

List of awesome Elixir and Erlang libraries

## Requirements
- Git
- GNU Make
- Docker >= 20.0

## Installation

```bash
$ git clone https://github.com/nunsez/awesome_elixir.git
$ cd awesome_elixir
$ make build
```

## Usage

### Start

```bash
$ make up

# then visit http://localhost:4000/
```

### Stop

```bash
$ make down
```

### Force sync

```bash
$ make sync
```
